# Python3 Blog Demo
## About Project
#### It's a demo of blog which based on Python 3.5.

---

## Environment
* Python 3.5 + 3rd party libs, etc.

## Project Structure

python3blog/   <-- 根目录

 - backup/               <-- 备份目录

 - conf/                 <-- 配置文件

 - dist/                 <-- 打包目录

 - www/                  <-- Web目录，存放.py文件
  
   -- static/            <-- 存放静态文件

   -- templates/         <-- 存放模板文件

 - README                <-- README文件
